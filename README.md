# Deel Frontend Challenge

## Getting Started

Run the follow comands to start project local:

```bash
- npm install
- npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

## Technologies

This project was built with:

```bash
- Vite
- React
- Typescript
- Css + Variables
```
